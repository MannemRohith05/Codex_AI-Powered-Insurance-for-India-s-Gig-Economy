# 🚨 HACKATHON JUDGES: PLEASE READ THIS FIRST 🚨

To ensure this project runs smoothly with all AI, Weather, SMS, and Payment integrations working out-of-the-box, you **must** use our pre-configured Environment Variables.

### What you need to do:
1. Open the **`.env.example`** file (or scroll to the bottom of the **`README.md`**).
2. Inside, you will find a **Google Drive link**.
3. Open that link to download our live **`.env`** file.
4. Place that downloaded **`.env`** file directly into the root directory of this project (`gigshield/.env`).

Once you've placed the file, simply run `npm run dev` and the entire project will work perfectly!

Thank you!
